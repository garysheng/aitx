# AITX Explorations — candidates, not goldens

Everything in this folder is **agent output pending human approval.** Nothing
here is blessed. It is where new molecules land so a human can look, react, and
decide.

**The promotion path:** `explorations/` → a human blesses it → move to
`goldens/`. A "golden" is a human-approved asset (see `../goldens/README.md`).
The agent never writes straight into `goldens/`.

Assets here still carry full provenance recipes, so once blessed they move over
reproducibly, unchanged.
