# AITX Goldens — a blessed library of base assets

This folder is where the AITX brand becomes **software you can compound**.

> **Golden means human-approved.** A file lands here only after a human blesses
> it. The agent never writes its own output straight into `goldens/` — candidates
> stage in [`../explorations`](../explorations/README.md) and get promoted here on
> approval. If it wasn't blessed, it isn't golden.

## Atoms and molecules

The AITX universe (`../universe`) holds the **atoms**: Golden Atomic Brand
References (GABRs), each locked and version-controlled.

- **Characters** — `michael-daigler`, `jake-oshea` (masters, faces, poses, backs), the `founders` two-shot.
- **The mark** — `aitx-mark` (the sacred orange open-arms logo; load-bearing on every render).
- **The place** — `antler-venue` (sign-in + speaking alcove plates).
- **The products** — `aitx-skylark` (the custom shoe), `aitx-merch` (cap, tote, tumbler, hoodie), the "Texas loves AI" tee back.

A **golden** in this folder is a **molecule**: a finished property — a flyer, an
Instagram post, a meme, a sticker — *composed from those atoms*. Every molecule
is generated by passing the relevant atoms in as references, so the mark always
renders as the real mark, the founders always read as themselves, and the venue
is always the real Antler.

```
flyer  =  aitx-mark  +  antler-venue  +  type
IG post =  founders two-shot  +  aitx-mark  +  headline
sticker =  aitx-mark  ( + Texas silhouette )
```

## Why version-control the brand

Because the brand is now generated by software, the reusable reference *is* the
source of truth. Keeping the whole thing in git means you can:

- **See the brand evolve** — every change is a commit; the diff is the changelog.
- **Iterate intentionally** — branch a new look, compare, keep or discard.
- **Imagine and merge** — compose new molecules from the same atoms and merge the
  ones that land, so alignment compounds instead of drifting.

This is the imagination economy applied to a brand: the atoms are blessed once,
and every new asset gets faster, cheaper, and more on-brand than the last.

## Contents

- `flyers/` — event posters (hackathon, monthly meetup)
- `social/` — Instagram / social posts
- `memes/` — shareable AITX memes
- `stickers/` — die-cut vinyl stickers (one sticker per file)

These are the seed set for an AITX asset generator (Claude Agent SDK): a minimal
tool that lets anyone in the community mint on-brand flyers, posts, and memes by
composing the same atoms.
